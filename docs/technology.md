# Technology Stack (PoC - Free Tier Optimized)

## Frontend & API
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (Modern, clean, "Anti-Corporate" aesthetic)
- **Deployment:** Vercel (Hobby Tier)

## Backend & Database
- **Database:** Supabase (PostgreSQL)
- **Vector Search:** pgvector (For matching member profiles via embeddings)
- **Authentication:** Supabase Auth (Magic Link or LinkedIn OAuth)

## AI Stack
- **Orchestration:** Vercel AI SDK
- **LLM:** OpenAI GPT-4o-mini (Cost-effective for synthesis and facilitation)
- **Embeddings:** text-embedding-3-small

## Video & Communication
- **Video SDK:** Daily.co (2,000 free minutes/mo) or Jitsi Meet (Open Source)
- **Email:** Resend (Free tier for transactional invites)
- **Scheduling:** Cal.com API (or custom simple slot-picker)
